📌 Project Overview
The Quick Commerce Sales and Delivery Intelligence Dashboard provides insights into sales performance, customer payment preferences, delivery efficiency, and order trends.
This dashboard helps businesses monitor key metrics and make data-driven decisions using interactive visualizations built in Power BI.

📊 Dashboards Included
1️⃣ Overview Dashboard
Provides a summary of key business metrics:
Total Orders
Total Revenue
Total Quantity Sold
Average Delivery Time
2️⃣ Sales Analysis Dashboard
Analyzes product and city-wise sales performance.
3️⃣ Payment Insights Dashboard
Shows customer payment preferences such as UPI, Card, and Cash.
4️⃣ Delivery Performance Dashboard
Evaluates delivery efficiency across different cities.
5️⃣ Order Trend Dashboard
Tracks order and revenue trends over time.
